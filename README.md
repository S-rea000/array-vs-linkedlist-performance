# Array vs Linked List Performance Comparison

This project compares the execution time of common operations on arrays and singly linked lists using the C programming language.

## Operations Covered
- Insertion
- Deletion
- Linear search
- Sorting (Bubble Sort for arrays)

## Input Sizes
- 10
- 100
- 1,000
- 10,000

## Tools
- C
- Standard C libraries
- Clock-based timing

## Notes
This implementation focuses on conceptual performance comparison rather than optimized benchmarking.
